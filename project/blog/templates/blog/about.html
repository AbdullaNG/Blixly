About Blixly

Blixly is a personal blog application built to demonstrate my skills in backend web development. This project is created using Django, and front-end technologies like HTML, CSS, and Bootstrap.

Key Features:

  • User Authentication: Secure login and registration for personalized user experience.
  • CRUD Operations: Create, read, update, and delete blog posts easily.
  • Responsive Design: A mobile-friendly interface designed with Bootstrap to ensure a seamless user experience on all devices.
  • Backend Powered by Django: The app is built using Django, leveraging its robust features for handling backend processes, including database management and user authentication.

Technologies Used:

The following technologies were used to build this blog application:

  • Backend: Django, Python
  • Frontend: HTML, CSS, Bootstrap

This app is both a personal project and a demonstration of my backend development skills. Feel free to check out the source code on GitHub.
